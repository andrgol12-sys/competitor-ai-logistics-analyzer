# 🚚 CompetitorAI — AI-анализатор сайтов транспортных компаний

Desktop-приложение для анализа сайтов конкурентов с использованием AI.

Проект автоматически:
- 🌐 парсит сайты конкурентов;
- 📸 делает скриншоты страниц;
- 🧠 анализирует контент через Vision + LLM;
- 📊 выделяет сильные и слабые стороны;
- 💡 предлагает рекомендации по улучшению;
- 📋 позволяет копировать результаты анализа.

---

## Возможности

### Анализ текста
- анализ описаний услуг;
- поиск сильных и слабых сторон;
- выявление УТП;
- генерация рекомендаций.

### Анализ изображений
- анализ визуального оформления;
- оценка структуры страницы;
- выявление UX/UI особенностей.

### Парсинг сайта
- автоматическая загрузка страницы;
- получение текста;
- создание скриншота;
- AI-анализ содержимого.

### Desktop-приложение
- интерфейс на PyQt6;
- копирование результатов;
- история запусков;
- сборка в `.exe`.

## Запуск

Создать окружение:

```bash
python -m venv .venv
```

Активировать:

```bash
.venv\Scripts\activate
```

Установить зависимости:

```bash
pip install -r requirements.txt
```

Запуск:

```bash
python run.py
```

Сборка:

```bash
cd desktop
python build.py
```
